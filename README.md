## üõ†Project Setup


 Clone the repository
```
https://github.com/stealth-bombeer/ApnaBaazar.git

```
To start the frontend user interface
```
cd client
yarn
npm dev
```
To deploy contract
```
cd server
yarn
yarn hardhat deploy --network seploia
```
